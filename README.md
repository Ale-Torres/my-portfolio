# Alejandro Torres – Portfolio Website

Live site: https://builtbyalex.vercel.app

This repository contains my personal portfolio website, where I showcase my projects, technical skills, and interests in software engineering and exploring new areas such as robotics.

## Overview

I am a UVic Software Engineering graduate with experience in full-stack web development. This portfolio highlights selected projects and serves as a central place to explore my work, technical background, and interests.

For the latest version of my work, please visit the live site. This repository contains the source code behind my website.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Git & GitHub
- Vercel (deployment)

## Key Features

- Modern, responsive UI built with Tailwind CSS
- Hero section with social links (GitHub & LinkedIn)
- About Me section with personal background
- Projects section showcasing technical work
- Continuous deployment via GitHub → Vercel
